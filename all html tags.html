HTML Tag:

1. html tag: is used to hold complete content of the web page
2. head tag: is used to indicate the head of the web page
3. title tage: is used to hold the title of the web page
4. body tag: is used to hold the actual content of the web page
5.P tag: or paragraf tag is used to hold multiline text content
6. br tag: is used a single line break
7. div tag: hold block content
8: span tag: inline elements to hold content in the same line
9. table tag: to render the content in the row and column way
10. ul and ol tag: to render the content in unordered or ordered way
11. list to display the list of the items
